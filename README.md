# Arduino Color Sorter

This project is an automated color sorting system built using Arduino. 
It detects colors using a TCS3200 color sensor and sorts objects using servo motors.

## 🔧 Components Used
- Arduino board
- TCS3200 Color Sensor
- 2 Servo Motors
- Connecting wires

## ⚙️ Working
- The color sensor detects Red, Green, and Blue values.
- Based on the detected color, the Arduino processes the data.
- The top servo scans the object.
- The bottom servo moves to a specific position to sort the object.

## 🎯 Features
- Automatic color detection
- Smooth servo movement
- Real-time color processing
- Efficient sorting mechanism

## 🚀 Applications
- Industrial automation
- Object sorting systems
- Educational projects
